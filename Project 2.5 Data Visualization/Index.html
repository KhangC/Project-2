Hello Classmates,

I began this project like many of you searching for a subject worthy of tackling. However, confronted by a question at work I realized that this project will be a great start in attempting to find an answer. Thus, the overall theme of this project is figuring out between two promotion which returns the best ROI. Answering this question required that I find the baseline sales and work from there. Yet, that was easier said than done. The next few passages will be how I went about analyzing, coding and charting my data.

Finding the baseline: I went about this with the quartile & outliers in mind. There were many articles published about how to obtain a basline sales but I found that quartile & outliers would benefit me more. Using the quartile & outliers and plotting a box and whiskers will allow me to figure out what outliers I need to eliminate in order to find the range of the revenue.

Coding: I gather the data from an online database and changed a few of the numbers to keep the data source confidental. Then I used Python to transfer the excel file over into a MySql data. I plan on using ajax and jquery to display my dataset. In terms of plotting, I use basic chartting code to display my visuals. /p>
